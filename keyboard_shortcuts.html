Keyboard Shortcuts To Be Used In GitHub

Site Wide Shortcuts:
S
To open search bar.

G + N
To open notifications.

ESC
Close hovercard.

CTRL + K
Opens command palette.








Repositories:
G + C
Open code tab.

G + I
Open issues tab.

G + P
Open pull requests tab.

G + A
Open actions tab.

G + B
Open projects tab.

G + W
Opens wiki tab.

G + G
Open discussions tab.








Source Code Editing:
CTRL + F
Search in file editor.

CTRL + G
Find next.

CTRL + Z
Undo.

CTRL + Y
Redo.

ALT + G
Jump to a line.

CTRL + SHIFT + G
Find previous.

CTRL + SHIFT + R
Replace.

CTRL + SHIFT + G
Replace all.








Source Code Browsing:
T
Open file finder.

L
Jump to a line.

W
Switch branch or tag.

Y
Expand a URL to its canonical form.

I
Show or hide comments on diffs.

B
Opeb blame view.
